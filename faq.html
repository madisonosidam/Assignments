Web Development Basic Terminology

Frontend (Client-side)
What the user sees and interacts with in their browser. This includes the layout, design, interactive elements, and content.
HTML (Hypertext Markup Language)
The standard markup language for creating web pages and defining their structure (headings, paragraphs, images, links, etc.).
CSS (Cascading Style Sheets)
A language used to style the visual presentation of HTML documents, controlling colors, fonts, layout, and responsiveness.
JavaScript
A programming language that enables interactive and dynamic content on web pages, such as animations, form validation, and complex user interfaces.
Responsive Design
A design approach that ensures a website adapts and looks good on various devices and screen sizes (desktops, tablets, mobile phones).
Framework (Frontend)
A collection of pre-written code and tools that provide a structure for building user interfaces more efficiently (e.g., React, Angular, Vue.js, Bootstrap).
Backend (Server-side)
The "behind-the-scenes" part of a website that handles data storage, server logic, databases, and communication with the frontend. Users don't directly interact with the backend.
Server
A computer program or device that provides functionality for other programs or devices (clients). In web development, it stores and serves web pages and data.
Database
An organized collection of structured information or data, typically stored electronically in a computer system. Common databases include MySQL, PostgreSQL, MongoDB.
Programming Language (Backend)
Languages used to write the server-side logic (e.g., Python, Node.js, PHP, Ruby, Java).
API (Application Programming Interface)
A set of rules and protocols that allows different software applications to communicate and exchange data.
Full-stack
Refers to developers who work on both the frontend and backend of web applications, possessing expertise in all layers of development.
CMS (Content Management System)
Software that helps users create, manage, and modify content on a website without needing extensive technical knowledge (e.g., WordPress, Drupal, Joomla).
Hosting
The service that makes your website accessible on the internet by storing its files on a server.
Domain Name
The unique, human-readable address of a website on the internet (e.g., example.com).
DNS (Domain Name System)
Translates domain names into IP addresses, which are numerical addresses that computers use to identify each other on the internet.
HTTP (Hypertext Transfer Protocol)
The foundational protocol for data communication on the World Wide Web.
HTTPS (Hypertext Transfer Protocol Secure)
A secure version of HTTP that encrypts data transmitted between a web browser and a website, ensuring privacy and security.
UI (User Interface)
The visual elements and interactive components of a website or application that users interact with (e.g., buttons, menus, forms).
UX (User Experience)
The overall experience a user has while interacting with a website or application, encompassing ease of use, satisfaction, and efficiency.
